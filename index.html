App Creation Cheat Sheet

Quick start guides for modern app development

Expo App

1. Install Expo CLI

npm install -g expo-cli

2. Create new project

expo init my-app

Choose a template (blank, tabs, etc.)

3. Navigate to project

cd my-app

Development with Expo Go

expo start

Scan QR code with Expo Go app or use emulator

Prebuild (for custom native code)

expo prebuild

Generates ios and android native directories

Development Builds (EAS Build)

First install EAS CLI:

npm install -g eas-cli

Login to Expo account:

eas login

Configure project:

eas build:configure

Build for development:

eas build --profile development --platform android eas build --profile development --platform ios

Uses your own credentials for native builds

Next.js App

1. Create new project

npx create-next-app@latest

Follow prompts to configure project

2. Navigate to project

cd my-app

3. Start development

npm run dev

Runs on http://localhost:3000

4. Build for production

npm run build npm start

Vite React App

1. Create new project

npm create vite@latest my-app -- --template react

For TypeScript: use --template react-ts

2. Navigate to project

cd my-app

3. Install dependencies

npm install

4. Start development

npm run dev

Runs on http://localhost:5173

Node.js Express App

1. Initialize project

mkdir my-app && cd my-app npm init -y

2. Install Express

npm install express

3. Create server file

touch server.js

Add this basic server code:

const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(PORT, () => {
  console.log(`Server running on http://localhost:${PORT}`);
});

4. Start server

node server.js
